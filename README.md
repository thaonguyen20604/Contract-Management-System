# CNPM-FINAL (Contract Management System)

A Windows Form application for managing contracts, clients, and related data.

## 🧩 Features
- Create, update, and search contracts
- Manage client and project information
- Import and export data via Excel/SQL
- Multi-layer architecture: **GUI – BLL – DAL – DTO**

## 🛠️ Technologies
- C#, .NET Framework (WinForms)
- SQLite / SQL Server
- Visual Studio 2022

## 🚀 How to Run
1. Clone this repository
2. Open `CNPM-FINAL.sln` in Visual Studio
3. Run the project (`Ctrl + F5`)
